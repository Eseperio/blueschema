# Vertical Menu Schema

Vertical navigation menu component for sidebars and side navigation.

## Properties

### `type` (required)
- Value: `"menuVertical"`

### `description` (required)
- **Type:** string
- **Description:** What menu items to display

## Example

```json
{
  "type": "menuVertical",
  "description": "Admin sidebar with Dashboard, Users, Posts, Settings, Logout links"
}
```
