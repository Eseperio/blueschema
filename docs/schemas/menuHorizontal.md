# Horizontal Menu Schema

Horizontal navigation menu component for top navigation bars.

## Properties

### `type` (required)
- Value: `"menuHorizontal"`

### `description` (required)
- **Type:** string
- **Description:** What menu items to display

## Example

```json
{
  "type": "menuHorizontal",
  "description": "Main navigation with Home, Posts, Categories, About, Contact links"
}
```
